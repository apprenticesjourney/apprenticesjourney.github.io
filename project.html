¿Cómo estoy haciendo esto?






How did this started?


This idea was born when I was back in Mexico after attended GitHub Satellite and Maintainerati at Berlin last May.

Esta idea nació cuando regrese a México, después de asistir a GitHub Satellite y a Maintainerati, en Berlín en el mes de mayo.

The Core Idea

I met a lot of interesting people at Berlin, so at this moment I understood that my mission was create something with this talented people! I wanted to go one step further than only make a project based on interviews or a podcast. I'm a software developer and a graphic recorder, so this project should be a mix of them.

Conocí muchas personas interesantes en Berlín, y en ese momento entendí que mi misión era crear algo con todas esas personas talentosas. Quería ir más allá de sólo hacer un proyecto de entrevistas o un podcast. Soy desarrollador de software y facilitador gráfico, este proyecto debía ser una mezcla de ambos.

The Learning System

The idea behind "The Apprentice's Journey" is to create a learning system. Each conversation is a dialectical exercise, through this, It could be possible to make a narrative analysis, as the Hero's Journal and model a system using the General Theory Systems.

La idea detrás de "El Camino del Aprendiz" es crear un sistema de aprendizaje. Cada conversación es un ejercicio dialéctico, a través del cuál será posible realizar un análisis narrativo, cómo el Camino del Héroe y modelar un sistema bajo la mirada de la Teoría General de los Sistemas.

My Motivations

I want to looking for the apprentice's journey because I'm an apprentice looking for new ways to discover the software world and I share it with everyone. I'm want to generate content from my country, Mexico, and with this say "Hey! Latin America are creating valuable content!"

Quiero encontrar el camino del aprendiz porque soy un aprendiz en busca de nuevos caminos para descubrir el mundo del software y compartirlo con todos. Quiero generar contenido desde mi país, México, y con esto decir "¡Hey! ¡América Latina esta creando contenido valioso!"
